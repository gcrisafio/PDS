# Proyecto de Medición con Acelerómetro Integrado en Android

## Descripción
Este proyecto consiste en realizar mediciones utilizando un acelerómetro integrado en un dispositivo Android. Las muestras fueron recibidas y procesadas por MATLAB como MobileSensorData en PC. El dispositivo Android tiene instalada la aplicación MATLAB Mobile, que permite realizar mediciones con los sensores integrados en el celular.

## Contenido
- **Archivos de datos: Excel** Contiene las muestras obtenidas del acelerómetro.
- ** Procesamiento Digital ** Gráficas y análisis obtenidos a partir de las mediciones.

## Instrucciones
1. **Recolectar datos:** Utiliza el acelerómetro del dispositivo Android para tomar mediciones con la aplicación MATLAB Mobile.
2. **Transferir datos a MATLAB:** Guarda las muestras en el formato MobileSensorData y transfiérelas a MATLAB en tu PC.
3. **Procesar datos:** Usa los scripts proporcionados para analizar y visualizar los datos.
